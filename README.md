
🎭 게임 요소별 특징  

환자 캐릭터 😆  
표정: 과장되고 코믹한 리액션  
스타일: 치료할수록 더 크게 반응  
예시: 드릴 사용 시 깜짝 놀라 눈이 커지거나 입을 크게 벌림  

치아 문제 🦷  
종류: 충치, 더러워진 치아, 과장된 치석  
스타일: 현실보다 훨씬 과장된 비주얼로 표현  

사운드 효과 🔊  
특징: 웃음을 유발하는 효과음  
스타일: 실제 소리보다는 장난스러운 효과 위주  

🎨 연출 및 디자인 방향  
각 요소는 다음과 같은 방향으로 설계됩니다:

- 카툰 스타일의 그래픽
- 굵은 외곽선과 선명한 색감
- 표정과 움직임이 강조된 애니메이션
- 공포 요소 없이 밝고 장난스러운 분위기 유지

🛠️ 기술 스택  
Frontend: React  
Styling: Tailwind CSS  
UI 구성: 컴포넌트 기반 구조  
Animation: CSS Animation / Placeholder Animation  

💡 개선 아이디어 5개  
1. 다양한 환자 캐릭터 추가 (어린이, 어른, 동물 캐릭터)  
2. 치료 성공 시 점수 또는 별점 시스템 도입  
3. 새로운 치과 도구 해금 시스템  
4. 타임 어택 모드 추가  
5. 치료 결과에 따른 엔딩 연출 추가  
 Changes are automatically pushed to this repository
4. Vercel deploys the latest version from this repository
